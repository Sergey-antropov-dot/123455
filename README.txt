# Random Password Generator

### Автор: Попыловский Михаил
### Дата: 07.05.2026

## Описание

Графическое приложение «Random Password Generator» позволяет генерировать случайные пароли с гибкими настройками:
- Выбор длины пароля (от 6 до 32 символов).
- Включение и отключение цифр, букв и спецсимволов.
- Просмотр и сохранение истории созданных паролей в файл JSON.
- Загрузка истории паролей из файла.
- Валидация: проверка длины и наличия хотя бы одного типа символов.
